HTML Basic Concepts

HTML (HyperText Markup Language) is the standard language used to create webpages and web applications. HTML defines the structure of a web page using various elements represented by tags.

1. Basic Structure of an HTML Document

An HTML document starts with the <html> tag and typically includes the following sections:

  • <!DOCTYPE html> – Declares the document type and version of HTML being used (HTML5 in this case).
  • <html> – Defines the root of an HTML document.
  • <head> – Contains meta-information, links to external resources (like CSS), and the page title.
  • <body> – Contains the visible content of the web page, like headings, paragraphs, images, etc.

2. Important HTML Tags

Below are some of the most commonly used HTML tags:

  • <h1> - <h6> – Header tags, where <h1> is the largest heading and <h6> is the smallest.
  • <p> – Defines a paragraph.
  • <a> – Defines a hyperlink, used for linking to other web pages or resources.
  • <img> – Used to display images on the page.
  • <ul> – Creates an unordered list, often used for bullet points.
  • <ol> – Creates an ordered list, often used for numbered items.
  • <li> – Defines list items, and is used inside both <ul> and <ol>.
  • <table> – Used to create tables, with <tr> (table row), <td> (table data), and <th> (table header).
  • <div> – A generic container tag that can be used to group elements for styling or scripting purposes.
